What Are Development Environments?

Development Environments are the software and tools used to create games. They are often paired with programming languages and have a lot of important features that allow game developers to actually make games.

Two main Development Environments are used by amateurs and professionals alike to make quality games: Unity and Unreal Engine.

Unity



Unity is developed by Unity Technologies and is used to make many different 3D and 2D games.

It uses the C# programming language and has many features for creating games and model renders.

Unreal Engine





Unreal Engine is developed by the video game company Epic Games.

It uses the C++ programming language, and has similar features and layout to that of Unity.

It has been used to create many popular games.
