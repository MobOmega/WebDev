What is Back-End Web Development?


Back-end development is the server side or functional side to web development. It's what makes things happen. Think of it as what runs the Google Search engine, whereas Front-end is how the page looks.

Back-end is transferring data and using functionality to make the experience for users more use and give it more substance. Back-end can run on basically any programming language.

What Pieces Are There To This Puzzle?


  • Databases
    • How the website fetches information
  • Servers
    • How the websitestores information and is accessed
  • APIs (Application Programming Interfaces)
    • How everything is tied together seamlessly
  • Scripting
    • Providing functionality and responsiveness directly to the user and transferring data
  • Middleware
    • Connecting the front-end and back-end, putting everything together
