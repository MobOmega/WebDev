What Is Programming?

Programming is everything in the online space. It's how we transfer data across the entire world, how we get and give new information. Without it, instant communication accross the entire world would be very hard to maintain. Programming is giving a computer a set of instructions. This site exists to teach you more about that.

Programming has been around for a long time, but it only became well-known and very important once we started making computers. Then, it was necessary to have a way to tell them what to do. Programming languages were made to give computers instructions in many different ways.

Programming exists in many forms. Whether you want to make a game, a website, or just make some program that acccomplishes some complicated task, programming is the tool for you.


This site contains resources for Web Development, Game Development, and general purpose Programming, all of which I have (some) experience in.
